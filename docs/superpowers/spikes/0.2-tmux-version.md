# Spike 0.2: tmux version ≥ 3.2 for `new-session -e`

**Date:** 2026-04-22
**Status:** ✅ PASS

## Evidence

```
$ tmux -V
tmux 3.6a

$ tmux new-session -d -s envtest -e FOO=bar 'sh -c "echo FOO=$FOO > /tmp/envtest.out; sleep 2"'
$ cat /tmp/envtest.out
FOO=bar
```

## Implication

`tmux new-session -e KEY=VAL` works as intended; `spec §6.3 (A)` spawn flow is viable with no fallback required.

## Follow-ups

- [x] Add a tmux version preflight in master startup (`tmux -V` parse; fail-fast if `< 3.2`).
- [x] Document `brew install tmux` in README prerequisites.
